# Startup Market Analyzer

## Overview
This project presents an interactive dashboard designed to analyze global startup market trends using Python and Streamlit. The dashboard enables users to explore how startup activity varies across countries, industries, and funding stages.

---

## Dataset
The dataset includes key startup information such as:
- Country
- Industry
- Funding Stage
- Total Funding
- Valuation
- Annual Revenue
- Success Score

---

## Features
- Interactive filters (Country, Industry, Funding Stage)
- Key performance indicators (KPIs)
- Dynamic charts and visualizations
- Data exploration in real time

---

## Dashboard Insights
The dashboard highlights several important patterns:
- Startup activity is concentrated in specific industries and countries
- Funding trends fluctuate over time
- Some industries show higher success scores and valuation levels
- Investment patterns differ across funding stages

---

## Visualizations
The dashboard includes:
- Top industries and countries (bar charts)
- Funding trends over time (line chart)
- Success score by industry
- Funding stage distribution
- Valuation and revenue comparisons
- Funding vs success correlation (scatter plot)
